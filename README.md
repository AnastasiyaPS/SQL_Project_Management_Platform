# SQL_Project_Management_Platform
Учебный SQL проект управления проектами и задачами. Платформа предназначена для осуществления удобного планирования, выполнения и контроля поставленных задач. В системе реализован функционал для управления задачами, ресурсами, назначением исполнителей и отслеживания статуса и сроков поставленных задач.
